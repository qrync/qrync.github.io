# QRync — Instant QR Code & Barcode Generator + Scanner

A free, browser-based QR code and barcode generator with a built-in scanner. No sign-up, no ads, no data stored — everything runs locally in your browser.

🔗 **Live Site:** [qrync.github.io](https://qrync.github.io)

---

## Features

**QR Code Generator**
- Custom dot styles: Square, Rounded, Dots, Classy, Classy Rounded, Extra Rounded
- Custom corner styles: Sharp, Rounded, Circle
- Full color customization — dot color, background color, transparent background
- Embed a logo (preset icons or your own image)
- Error correction levels: Low, Medium, High, Max

**Barcode Generator**
- Formats: Code 128, EAN-13, UPC-A, Code 39, ITF-14, MSI
- Customizable bar color, background, text size, and bar height

**Scanner**
- Scan QR codes and barcodes via camera or image upload
- Supports Code 128, Code 39, EAN-13, EAN-8, UPC-A, UPC-E, ITF, Codabar, and more

**Download Options**
- PNG, SVG, and PDF export

**PWA Support**
- Install as a native app on Android and iOS
- Works offline after first visit

---

## Usage

No installation needed. Just open the site and start generating or scanning.


---

## Privacy

Everything runs locally in your browser. No content you enter, no QR code data, and no scanned results are ever sent to any server. There are no analytics, no tracking, and no cookies.
